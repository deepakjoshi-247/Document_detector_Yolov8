# Real-Time Document Detection Model
This project uses YOLOv8 to detect and classify Pan Cards and Aadhaar Cards (Front/Back) via a live webcam feed.

## Features
- Real-time Detection: Handles live video inference.
- Multi-Class Support: Detects card borders and specific fields (Name, DOB, etc.).
- High Accuracy: Validated with a mAP50 of 0.85.

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run the detection: python main.py

## Note - 
1. For now the accuracy is 85%.
2. Our Future Roadmap is now to increase the accuracy and To make it detect more Documents.
